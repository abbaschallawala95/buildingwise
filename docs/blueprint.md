# **App Name**: BuildingWise

## Core Features:

- Building Management: Enable admin to create, read, update, and delete buildings and manage their details, with isolated data per building.
- Member Management: Enable admin to add, update, and delete members with details such as full name, flat number, contact, monthly maintenance, and previous dues.
- Maintenance Collection: Collect maintenance payments, adjust previous dues, generate unique receipt numbers, and save transactions.
- Extra Collection: Create and manage extra collections like light bills, divide amounts equally among members, and generate individual receipts.
- Expense Management: Add, edit, and delete expenses, categorize them, and track expense history, linking them to specific buildings.
- Dashboard: Display key metrics such as total members, maintenance collected, extra collections, total expenses, pending dues, and a net balance chart.
- Automated Receipt Generation and Distribution: Automatically generate PDF receipts and suggest personalized WhatsApp messages. A tool could assist the user by suggesting what information would be helpful in the WhatsApp message.

## Style Guidelines:

- Primary color: Moderate cyan (#4FC3F7) for a modern and reliable feel.
- Background color: Very light cyan (#E0F7FA).
- Accent color: Violet (#9575CD) for highlighting key actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif, which provides a modern and neutral look for both headlines and body text.
- Use flat, minimalist icons to represent different functions and modules.
- Maintain a clean, grid-based layout with clear sections for different modules. Ensure responsive design for various screen sizes.
- Use subtle animations to provide feedback on interactions and transitions, enhancing the user experience.
